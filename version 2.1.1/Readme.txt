Dedescription:
	Pacman is a fun game. In this game there are three monsters,
	pacman, and predefined maps with points. The pacman gets points
	as he keeps on eating the points without coming in contact with
	any other monster. The monsters have a vissibilty in the shape
	of a '+' shape with the monster as the center, throughout the 
	maps. These monster repel each other if travelling in the same
	direction.
	Controls :
		up		:	up arrow
		down		:	down arrow
		right		:	right arrow
		left		:	left arrow
		pause		:	p
		unpause		:	p
		save and exit	:	y (while paused)\
		exit		:	n
